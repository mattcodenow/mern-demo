# MERN Stack

## Create Project Directory
From the terminal:
```
mkdir mern-demo
```

## Open Project Directory In Visual Studio Code
The following will open up a new window with the new folder loaded as a new project.  
From the terminal:
```
code ./mern-demo
```

## Initialize NPM
From the terminal inside the new VSC window:
```
npm init -y
```

## Initialize Git
From the terminal:
```
git init
```

## Create GitIgnore 
From the terminal:
```
touch .gitignore
```

## Edit GitIgnore
Add the following content to the new .gitignore file:
```
node_modules
notes

```

## Create README.md
From the terminal:
```
touch README.md
```

## Install TypeScript Tools
```
npm i -D typescript ts-node nodemon @types/node
```
